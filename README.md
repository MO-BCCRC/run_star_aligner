###Dependencies

###Usage
It runs STAR aligner. Please refer to its user manual (https://github.com/alexdobin/STAR/blob/master/doc/STARmanual.pdf) and the paper (http://bioinformatics.oxfordjournals.org/content/29/1/15) for more info.
